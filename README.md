# Датасет
Использовался датасет S&P 500 с 2015-05-01 по 2025-05-01. Для проверки используется тот же S&P с 2025-05-01 по 2026-05-01

# Модель
Готовая предтренированная модель лежит в models/scaler.pkl и models/vae_trading.pth. Вариационный автоэнкодер (VAE) для вероятностного моделирования и генерации финансовых временных рядов. Проект решает задачу прогнозирования поведения рынка через оценку плотности распределения, а не предсказание точечной цены

## Постановка задачи

На основе исторических данных создать модель распределения и модель генерации данных, реализовать вариационный вывод и визуализировать плотность распределения предсказаний

## Архитектура и решения

* Модель распределения (Encoder): нейросеть сжимает исторические окна рыночных данных (цены и объемы) в параметры скрытого вероятностного пространства
* Модель генерации (Decoder): нейросеть берет значения из скрытого пространства и разворачивает их обратно в новые, синтетические, но реалистичные рыночные сценарии
* Вариационный вывод: обучение модели происходит через балансировку ошибки восстановления данных и правильного формирования скрытого пространства
## Результаты
1. Оценка плотности: модель успешно выучила дисперсию рынка. Графики ядерной оценки плотности на тестовых данных доказывают, что сгенерированные предсказания математически совпадают с реальной волатильностью рынка
2. Симулятор Монте-Карло: разработана вероятностная модель предсказаний. Скрипт генерирует 1000 альтернативных вариантов рынка на месяц вперед и автоматически рассчитывает ожидаемую доходность, вероятности профита и риски просадки для вложенного капитала
## Структура проекта
- models/vae_model.py - архитектура сети (Encoder, Decoder, Loss-функция)
- utils/data_loader.py - загрузка данных, расчет лог-доходностей, генерация временных окон
- train.py - скрипт обучения
- evaluate.py - валидация плотности распределения и запуск симуляции Монте-Карло
